National News Site

In den folgenden Aufgaben geht es darum, eine Webseite einer Zeitung zu erstellen und verschiedene Inhalte darin einzubinden.

  1. Verwenden Sie die Vorlage: template/index .html und die textuellen Inhalte: media/as-text .txt. Die Vorlage enthält das Grundgerüst sowie einige Styles (styles/style.css) um ein minimales Layoutvzu erzeugen für Mobil & Desktop. Kümmern Sie sich nicht um die Darstellung, sondern nur um die Struktur des HTML. So soll die Seite schlussendlich aussehen
  2. Alle Medien wie Videos oder Bilder inklusive den SVG-Icons und dem Bookmark-Icon finden Sie im Ordner media.
  3. Erweitern Sie die Vorlage um weitere Abschnitte wie Sections, Aside oder Navigationen.
  4. Erweitern Sie das Dokument um Charset, Titel und Metadaten. Wählen Sie einen Titel, der das Suchmaschinenranking positiv beeinflusst und dem Benutzer hilft, die Seite einzuordnen. Ergänzen Sie Metadaten wie Autor, Publisher, Keywords, Description und Bookmark icon (icon-national.png).
  5. Bauen Sie alle Inhalte gemäss Muster auf. Hinweise:
    • Verwenden Sie als Links für die Breadcrumb und die Hauptnavigation die gleiche Linkstruktur wie sie optisch sichtbar ist. Z.B. 
      /news/national/asien-ozeanien/
      oder 
      /news/front/latest/
      .
    • Benutzen Sie eine Datalist für die Suchvorschläge des Suchfeldes.
    • Fügen Sie das Logo als SVG direkt ins Dokument ein. Binden Sie alle andern SVG's wann möglich als Bild ein. (Für Interaktivität ist dies nicht möglich).
    • Benutzen Sie für das Such-Icon eine SVG-Data-Url. Passen Sie width und height des SVG's auf 3mm an. Codieren Sie den Inhalt (Ohne die XML-Deklarationszeile) des SVG's als Base64 mittels base64encode.org/.
    • Wenn Sie das time Element einsetzen, setzen Sie auch das datetime Attribut. Selfhtml Wiki: Time Element
    • Binden Sie für das Video die englischen Untertitel ein. Chrome kann diese noch nicht anzeigen. Benutzen Sie Firefox um dies auszuprobieren.
    • Nutzen Sie figure für alle Elemente mit Unterschriften. Setzen Sie das alt-Attribut für alle Bilder. Nutzen Sie caption für die Tabelle. Nutzen Sie small Für Angaben zum Autors. Sie können die Angabe über den Autor als fertiges HTML von der Mediawiki Seite erhalten. Folgen Sie dazu dem Link und klicken Sie auf "Download". Bei "Credit the author" finden Sie die entsprechenden Snippets.
    • Fügen Sie den Formularelementen sinnvolle Validierung hinzu.
    • Für das Youtube Video erhalten Sie den Embed-Code auf Youtube unter "Share".
    • Schauen Sie sich die folgende Liste an und finden Sie heraus, für welche Elemente sie WAI-ARIA Attribute vergeben sollen: w3.org/TR/html-aria/#docconformance. Tipp: Schauen Sie sich vor Allem die Elemente an, wo "No corresponding role" steht.
  6. Benutzen Sie den W3C Validator um ihre Seite zu validieren.

Vorgehen

  • Bekanntes zuerst
  • Von Aussen nach Innen
  1. Document Einleitung/Kopfbereich (header) & Abschliessendes/Fussbereich (footer).
  2. Inhalt Aufteilung: Navigation (nav), Hauptinhalt (main), Zusatzinhalte (aside).
  3. Ev. Kopfbereiche für Navigation, Hauptinhalt und Zusatzinhalt.
  4. Hauptinhalt Aufteilung: Kapitel/Thematische Abschnitte (section's).
  5. Kapitel Aufteilung: Kopfbereich (header), Beiträge/Artikel (article's).
  6. Artikel Aufteilung: Kopfbereich (header), Inhalte (p, figure, table, ...), Fussbereich (footer).
  7. Kopfbereiche können aus mehreren Titeln bestehen. Z.B. einem Haupttitel (h2, h3, ...) und einem Nebentitel (hx+1) oder einem Teaser (p).
  8. Fussbereiche können Rechtliche Informationen (small), Kontaktinformationen (address), Veröffentlichungsinformationen (time), ... enthalten.
