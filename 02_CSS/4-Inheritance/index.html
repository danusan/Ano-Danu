⇦ WED1 - CSS Vererbung und Kaskadierung

Theorie

Lesen Sie die Artikel http://wiki.selfhtml.org/wiki/CSS/Vererbung und http://wiki.selfhtml.org/wiki/CSS/Kaskade um Vererbung, Kaskadierung und Spezifität von Regeln zu verstehen.

Aufgabenbeschreibung

Sehen Sie sich die Darstellung im Abschnitt Vorlage an. (ganz unten) Untersuchen Sie mittels Entwicklertools die definierten Styles und beantworten Sie die Fragen. Lösen Sie anschliessend die Aufgaben.

Fragen

  1. Warum werden die Unterpunkte der Liste ebenfalls blau eingefärbt, obwohl explizit nur die Oberpunkte eingefärbt werden?

    Die Eigenschaft "color" vererbt standartmässig ihren Wert auf Kindelemente. Darum führt eine Definition der Farbe für das Elternelement dazu, das auch die alle Unterpunkte blau eingefärbt werden. 

    body > ul > li {
	color: rgb(0, 96, 160);
}

  3. Erklären Sie, warum der Listenpunkt "Hauptsitz" fett dargestellt wird.

    Da der Inline-Style den vererbten Wert überschreibt.

    Wichtig: Die Spezifität hat bei diesem Punkt keinen Einfluss!

  4. Erklären Sie, warum der Listenpunkt "250 Mitarbeiter" nicht kursive dargestellt wird. Berechnen Sie die dazu die Spezifität des Inline-Styles und des im Stylesheet definierten Styles.

    Siehe Spezifität

    • Spezifität der Regel des Inline Styles (element { font-style: normal; })
      • S:1 (Style-Attribut)
      • A:0 (ID-Selektoren)
      • B:0 (Attribut-, Klassen- und Pseudoselektoren)
      • C:0 (Typ- oder Pseudoelementselektoren)
      Total: 1 0 0 0
    • Spezifität der Regel im Stylesheet (li > ul > li { font-style: italic; })
      • S:0 (Style-Attribut)
      • A:0 (ID-Selektoren)
      • B:0 (Attribut-, Klassen- und Pseudoselektoren)
      • C:3 (Typ- oder Pseudoelementselektoren)
      Total: 0 0 0 3

    Da die Spezifität der Regel im Inline Style höher ist als die Spezifität der Regel aus dem Stylesheet, wird das Element 'normal' dargestellt.

Aufgaben

  1. Färben Sie alle Unterpunkte in der Vorlage mit nur einer Regel schwarz ein.
    2. ul > li > ul {
	color: black;
}
  2. Die Unterpunkte sollen ebenfalls quadratische Aufzählungspunkte verwenden. Definieren Sie eine entsprechende Regel für 
    li > ul
    ohne das Wort "square" zu verwenden.

    3. "inherit" erlaubt es, den Aufzählungspunkttyp vom Elternelement zu erben:

    li > ul {
	list-style-type:inherit;
}
  3. Binden Sie ein zusätzliches Stylesheet VOR dem Stylesheet style.css ein, in welchem Sie eine Regel definieren, die die Unterpunkte ebenfalls fett formatiert und die definierte Regel im "default.css" überlagert. Verwenden Sie dazu NICHT "!important" und nicht "bold".

    Um die bestehende Regel zu überschreiben, muss eine Regel mit grösserer Spezifität erstellt werden:

    <link rel="stylesheet" type="text/css" href="myStyle.css">

    myStyle.css:

    body > ul > li ul {
	font-weight: inherit;
}
