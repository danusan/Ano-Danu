CSS Definieren

Nutzen Sie für diese Aufgabe die Vorlage files/csseinbinden.html

Die Vorlage enthält einen Ausschnitt von https://de.wikipedia.org/wiki/Cascading_Style_Sheets#Syntax

Ihre Aufgabe ist es, diesen Ausschnitt dem Style von der Wikipedia-Seite anzugleichen. Schauen Sie sich auch das Drucklayout der Wikipedia Seite an, was verändert sich?

  • Erstellen Sie 3 Stylesheets
    • screen.css: Enthält exklusive Styles für die Darstellung auf einem Bildschirm.
    • print.css: Enthält exklusive Styles für den Druck.
    • all.css: Enthält allgemeine Styles.
  • Binden Sie die Stylesheets korrekt ein.
  • Definieren Sie die Styles.

Hinweise

  • Nutzen Sie die Debug Tools um die Fonts / Farben von Wikipedia zu kopieren.
  • Definieren Sie als erstes die Screen-Styles; Danach die Print Styles und am Ende extrahieren Sie die Gemeinsamkeiten.
  • Ignorieren Sie das "Code highlighting".
