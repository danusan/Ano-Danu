Beantworten Sie folgende Fragen:

  1. Welche Farbe wurde für den Hintergrund der Überschrift dieses Dokumentes gesetzt? (h1)

    #0060A0 oder rgb(0, 96, 160)

  2. Wie werden die abwechselnden Aufzählungszeichen der Listenpunkte erzeugt?

    ol > li:nth-child(2n) Selektiert jedes zweite Listenpunktelement

  3. Welches Property ist dafür verantwortlich das dieser Text unter dem Bild steht?

    Das Herumfliessen wird durch das clear beendet. Somit fliessen dieses und alle folgenden Element nicht um das Element herum sondern beginnen erst danach.

  4. Wie wurde die Nummerierung der Überschriften zweiter Stufe (h2) realisiert?
    Für die Anzeige der Nummerierung wird die CSS-Counter Funktion benutzt. Diese erlaubt das definieren eines Counters, der inkrementiert und resetet werden kann. Reset-Punkt des Counters: Bei Überschriften zweiter Stufe muss dies z.B. jedes Mal geschehen, wenn eine Überschrift erster Stufe erfolgte. 
    body {
	counter-reset: {countername}; // setzt den Counter zurück
}
h2::before {
	counter-increment: {countername}; // erhöht den Counter
	content: counter({countername}); // gibt den Wert des counters aus.
}
    Der Wert des Counters wird mittels ::before als Pseudoelement vorne an den Titel angehängt.
